# ComposeMultiplatformImdbDemo — iOS

iOS client app consuming the [composemultiplatformsdk](https://github.com/erikg84/composemultiplatformsdk). The SDK provides all 8 screens as **SwiftUI Views auto-generated inside the XCFramework** — the client owns navigation and credentials.

## Technology Stack

| Layer | Technology | Why |
|-------|-----------|-----|
| **UI** | SwiftUI | Native iOS navigation (TabView + sheet drawer) wrapping SDK's Compose screens |
| **SDK Screens** | SKIE Sub-Plugin | Generates `public struct HomeScreen: View` inside ImdbSDK.framework — zero wrapper code in this repo |
| **iOS Bridging** | SKIE 0.10.11 (Touchlab) | Kotlin Flow to AsyncSequence, suspend to async, sealed to enum |
| **Compose Rendering** | Skia/Metal via ComposeUIViewController | SDK's Compose UI renders natively via Metal — SwiftUI hosts the UIViewController |
| **Project Generation** | XcodeGen | `project.yml` generates `.xcodeproj` — no committed Xcode project files |
| **SDK** | ImdbSDK.xcframework | Local framework (built from SDK repo) or via Gitea Swift Registry |

## Navigation

**Bottom tabs** (TMDB REST screens):
Home, Movies, TV, Search, Trending

**Sheet drawer** (GraphQL screens, via hamburger menu):
Countries, Continents, Languages

## How It Works

```swift
// AppCoordinator.swift — SDK screens are SwiftUI Views from the framework
import ImdbSDK

TabView(selection: $selection) {
    HomeScreen()       // auto-generated by SKIE sub-plugin
    MoviesScreen()     // no wrapper code anywhere
    TVShowsScreen()
    SearchScreen()
    TrendingScreen()
}

// Drawer destinations — also from the SDK
CountriesScreen()   // GraphQL
ContinentsScreen()  // GraphQL
LanguagesScreen()   // GraphQL
```

## Setup

1. **Build the SDK XCFramework** (one-time):
   ```bash
   cd /path/to/composemultiplatformsdk
   ./gradlew :imdb-sdk:assembleImdbSDKReleaseXCFramework
   ```
2. **Copy the framework**:
   ```bash
   cp -R imdb-sdk/build/XCFrameworks/release/ImdbSDK.xcframework \
     /path/to/ComposeMultiplatformImdbDemo-iOS/Frameworks/
   ```
3. **TMDB credentials** in `tmdb-token.txt` and `tmdb-api-key.txt` (gitignored)
4. **Generate Xcode project**:
   ```bash
   brew install xcodegen
   xcodegen generate
   open ComposeMultiplatformImdbDemo.xcodeproj
   ```
5. Build and run on simulator (iOS 16+)
